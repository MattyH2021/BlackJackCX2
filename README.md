Blackjack CX2

A lightweight blackjack card game for the TI-Nspire CX II calculator.

Built in C using the Ndless SDK and SDL, this project is designed to run as a native calculator game with a simple casino-style interface and optimized code-drawn graphics.

Features
Play-money blackjack gameplay
Hit, Stand, and Double
Split support
Insurance support
Dealer AI
Basic win/loss/push tracking
Code-drawn card graphics
Optimized for the TI-Nspire CX II 320x240 screen
No external image files
No 52-card sprite sheet required
Controls
ENTER        Confirm / deal / selected action
LEFT/RIGHT   Change bet or selected action
H            Hit
S            Stand
D            Double
P            Split
I            Insurance Yes
N            Insurance No
R            Reset from title/game over
ESC          Quit
